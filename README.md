# 만다라트 기반 SNS 서비스

(로고 및 사진 추가 예정)

## 💡 프로젝트 소개
만다라트 작성과 실천을 간편하게 만들고, 이용자들간 만다라트 및 관련 게시글을 공유함으로써 더욱 긍정적인 효과를 야기하는 서비스

## 📅 개발 기간
2023.10.17(화) ~ 2023.11.03(금)

## 🔨 기술 스택 

### FrontEnd
React (Webpack) (추가 예정)

### Backend
Django (추가 예정)
